## 8. Security Architecture

- No credentials in code вҖ” all secrets in `.env`
- Power Automate URL contains SAS token вҖ” treated as secret
- Contact deletion requires password authentication
- File upload validation: `.xlsx` only, 50MB max
- In-memory session data вҖ” no persistent user data beyond contacts
- Deploy behind reverse proxy (Nginx) for production HTTPS
- Single-user design вҖ” no authentication layer needed
